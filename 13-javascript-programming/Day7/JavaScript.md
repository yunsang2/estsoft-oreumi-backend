JavaScript 제어문

        {조건문}

            [if else 구문]
             조건이 참인지 거짓인지에 따라 참이면 절차를 수행
             거짓이면 else 이후에 절차를 수행
                
                if (조건식) {
                    참일시 실행;
                }
                else {
                    거짓일시 실행;
                }

        {중첩 조건문}

            [if else if 구문]
             조건문에 조건문을 중첩한 경우

                if (조건식) {
                    참일시 아래 조건식을 실행
                    if (조건식) {
                        참일시 실행;
                    }
                    else {
                        거짓일시 실행;
                    }
                }
                else {
                    거짓일시 실행;
                }

        {반복문}
        
            [while 구문]
             가장 기본적인 반복문
             주어진 조건식이 참인 동안 계속해서 실행 거짓이면 반복문 종료

                while (조건식)
                    참인동안 실행;

            [do while 구문]
             코드 블록이 먼저 실행하고, 나중에 조건을 검사하는 반복문
             코드 블록을 최소한 한 번은 실행해야하는 경우 사용

                do
                    참일시 실행(하지만 먼저 실행)
                while (조건식);

            [for 구문]
             while 구문은 조건에 비중을 두지만 for 구문은 반복횟수에 비중을 둔다

                for (초기화; 조건문; 실행후 실행되는 식)
                    참일시 실행;

    <배열>
     저장 공간들, 변수들의 집합
     각 저장공간을 원소 라고 하며 원소에는 인덱스를 이용해 접근한다
     선언할때는 [] 를 사용해 선언하며 자료형은 객체(object)이다

        var numbers = [4, 5, 4, 5, 3, 2];
        numbers[0] = 4, numbers[1] = 5, ..., numbers[5] = 2

        배열의 원소
         배열에서 값을 저장하는 각각의 저장공간
        
        배열의 인덱스
         배열의 각 저장공간, 즉 원소에 접근하기 위해 사용하는 정수
