JavaScript
 HTML 문서에서 사용하는 프로그래밍 언어
 정적인 HTML 문서에 동적인 기능 부여
 HTML 문서에서 사용하는 프로그래밍 언어
 스크립트언어이며 웹 브라우저의 동작을 제어한다
 최근에는 컴퓨터 애플리케이션에 못지 않은 웹 애플리케이션으로 발전
 브라우저에서 발생하는 다양한 이벤트에 연결할 수 있으며, 브라우저가 제공하는 기능보다 더 다양한 기능을 제공

 <head>요소의 <script> 요소가 마지막에 들어가며
 <script>요소를 통해 JavaScript 프로그램 작성 및 파일 연결

 src 속성으로 파일을 불러오면 해당 <script>요소안에는 프로그램을 작성할 수 없다

JavaScript 용어와 규칙

    표현식
     값을 만들어 내는 프로그램 코드

    문장
     프로그램을 구성하는 기본 단위
     문장은 반드시 ; 으로 끝내야함
     문장은 차례대로 실행. 위에 나온 문장의 실행이 끝나야 다음 문장이 실행

    키워드
     프로그램에서 미리 정의된, 큭별한 의미를 가진 단어

    식별자
     프로그램에서 이름을 붙일 때 사용하는 단어
     변수명과 함수명이 이 식별자에 해당한다
        영문자, 숫자, _, $ 조합해서 작성
        숫자로 시작할수 없다
        공백문자는 포함될수 없다
        대문자와 소문자를 구분한다
        키워드는 식별자로 사용할 수 없다

    주석
     프로그램 진행에 영향을 주지 않는 수식어

    괄호 작성 규칙
     (), {}, []
     모든 괄호는 쌍으로 이루워져있으며 짝이 안 맞으면 오류가 발생한다

    공백 문자
     빈칸, 개행문자, 탭 문자 등있고
     구분자로 쓰이며 공백문자 여러개는 무시된다

    함수와 메서드
     입력을 받아 작업을 수행하고 결과를 내보내는 프로그램 코드의 집합
     함수나 메서드를 실행시키는 것을 "함수를 호출한다" 라고 표현하며 호출하면서 필요한 값을 전달할 수 있는데, 이를 인자 라고한다

    자료형
     프로그램에서 다루는 값의 종류
     자료형에 따라 프로그램에서 값을 다루는 방법이 달라진다



JavaScript 객체와 메서드, 함수

    document 객체
     DOM을 나타내는 객체
     .뒤에 write 메서드로 DOM에 지정된 값을 출력한다

        write 메서드
         DOM 에 지정된 값을 출력하는 기능
         해당 메서드로 출력한 내용은 요소가 포함된 HTML 요소의 내부 마크업이 된다

    console 객체
         .뒤에 log 메서드로 브라우저 콘솔에 지정된 값을 출력한다
         (브라우저 콘솔은 DOM에 표기 되지 않으며 콘솔 에서만 출력된다)

        log 메서드
         브라우저 콘솔에 지정된 값을 출력하는 기능
         주로 디버그 메세지 등을 출력하기 위해 사용한다 

        group 메서드
         브라우저 콘솔에 출력하는 내용을 그룹으로 묶음
         작성후엔 groupEnd()를 작성해야한다
         ()안에 원하는 명칭을 적어 표기할수있다

        groupCollapsed 메서드
         브라우저 콘솔에 출력하는 내용을 그룹으로 묶은상태로 닫아놓는다
    
    
JavaScript 자료형, 상수와 변수
    자료형
     프로그램에서 다루는 값의 종류
     자료형에 따라 프로그램에서 값을 다루는 방법이 달라진다

        상수
         프로그램에서 사용하는, 변하지 않는 값
         대입을 통해 상수에 이름을 붙일 수 있다

        변수
         변하는 값, 값을 저장하기 위한 저장공간에 붙이는 식별자를 말한다
         프로그램에서 일시적으로 필요한 값을 일시적으로 저장할 목적으로 사용
         var 키워드로 선언하며, 대입 연산자(=)로 값을 대입한다.
         변수를 선언할때 값을 대입하는것을 변수의 초기화 라고 한다

        배열
         유형이 같은 여러 값들을 저장하기 위한 저장공간들, 즉 변수들의 집합
         배열은 객체(object)의 한 종류이다 각 저장공간은 원소 라고 하며 0부터 1씩 순서대로 증가한다
         배열의 원소들의 개수를 배열의 크기(length) 라고 한다
         마지막 인덱스는 (length - 1)이다
         배열은 배열 리터럴([])로 생성하며, 변수에 대입해서 접근한다
         또한 다른 자료형을 추가하거나 대입할 수 있다(c++이랑 다름)
            ex) var num = [4, 5, 4, 5, 3, 2];
            num[0] = 4, num[1] = 5, ..., num[5] = 2


        [string]      문자열
         ('', "")로 묶어서 표현 가능하지만 되도록 같은 따옴표를 사용
         (+)으로 문자열을 연결해 출력할수있다 하지만 숫자랑 문자열은 더할 수 없다
         피연산자들중 하나라도 문자열이면 다른 자료형을 문자열로 변환후 연결한다

        [number]      숫자
         특별한 값

            Infinity    무한대(숫자를 0으로 나누면 연산결과가 무한이 나옴)

            NaN         숫자로 나타낼수 없는 값을 의미하는 키워드

        [boolean]     불린(참, 거짓)

        [function]    함수
         일정한 기능을하는 역할을 가진 기계와 같다
         function 식별자() { 기능; } 라는 형태로 작성
         함수 호출 연산자'()'는 자료형이 함수인 형태뒤에만 붙는다
         함수 호출 표현식을 통해 실행된다

        [object]      객체

        [undefined]   선언되지 않았거나 할당되지 않은 값
         변수로 선언하지 않은 식별자
         변수로 선언을 했지만 초기화하지 않은 변수



JavaScript 연산자
    연산자
     

    표현식
     값을 만들어 내는 코드
     연산자와 피연산자로 이루어진다
     표현식은 항상 결과 값을 가진다, 즉 연산자는 항상 연산 결과를 가진다

        {typeof} 연산자
         typeof 연산자로 값의 자료형을 확인할 수 있다
         typeof 연산자의 연산 결과는 문자열이다

        [대입 연산자] (=)
         변수와 같은 저장공간에 값을 저장하는 연산자
         우변에 있는 값을 좌변에 있는 변수와 같은 저장공간에 대입한다

        [산술 연산자]
         평소 사용하는 계산 술식들( +, -, *, /, %, ** )
            5 % 3 = 2, 6 % 2 = 0
            나머지 연산은 나누고 나머지인 값이 연산결과가 된다

        [복합 대입 연산자]
         산술 연산자와 대입 연산자가 복합된 연산자이며 대입 연산자의 일종이다
         대입 연산자와 같이 좌변의 값은 변수 등 대입 가능한 값이 와야한다
         연산결과를 좌변에 다시 대입한다
         (+=, -=, *=, /=, %=, **=)
         ex) x = x += 4 
            x에 4만큼 더해서 다시 x에 대입

        [단항 연산자]
         피연산자가 하나인 연산자
            +num            변수의 값을 양수로 변환 
            -num            변수의 값을 음수로 변환 
            ++num, num++    변수의 값에 1을 더함, 식이 연산된 후 계산
            --num, num--    변수의 값에 1을 뺌  , 식이 연산된 후 계산

        [이항 연산자]
         피연산자가 두 개인 연산자

        [삼항 연산자]
         피연산자가 세 개인 연산자

        [전위 연산, 후위 연산]
         "전위 연산"은 문장에 실행이 되기전에 실행된다
            ++num 되어있으면 먼저 수행후 실행

         "후위 연산"은 문장의 실행이 끝나고 실행된다
            num++ 되어있으면 먼저 실행후 수행

        [비교 연산자]
         이항 연산자이며 양쪽에 있는 피연산자의 값을 비교
         '<', '>' 등등 크기 비교는 알고있는것과 일치한다
            ==          서로의 값이 같은지 비교
            !=          서로의 값이 다른지 비교
            ===         서로의 자료형이 같은지 비교 같으면 값을 비교
            !==         서로의 자료형이 다른지 비교 다르면 값을 비교

        [논리 연산자]
         피연산자를 참 또는 거짓으로 다루고, 연산 결과도 참 또는 거짓이 되는 연산자
            !       피연산자가 참이면 거짓, 거짓이면 참(단항 연산자)
            &&      피연산자가 모두 참이면 참, 아니면 거짓(이항 연산자)
            ||      피연산자 중에서 하나라도 참이면 참, 모두 거짓이면 거짓(이항 연산자)

        [조건 연산자]
         조건에 따라 다른 표현식을 연산 결과로 취하는 연산자
         피연산자가 3개다, 참이면 2번째 피연산자가 결과가 되고, 거짓이면 3번째 피연산자가 결과가된다
            (참 or  거짓) ? 참일때 연산결과 : 거짓일때 연산결과

    연산자 우선순위
        산술
        비교
        논리
        대입

    피연산자 우선 순위
        단항
        이항
        삼항

     우선순위가 같은 연산자들이라면 진행방향에 따라 우선순위가 정해진다

JavaScript 제어문
    제어문
     프로그램의 실행 흐름에 영향을 주는 문장

        {복합문}
         중괄호{}로 묶어 놓은 것, 블록 이라고도 한다
         문장 하나가 들어가야할 자리에 여러 문장들을 넣어야 하는 경우 사용한다
        
        조건문
         만약에 ~ 하면 ~ 하겠다 라는 식의 구문을 조건문이라한다

            [if 구문]
             조건이 참인지 거짓인지에 따라 참이면 절차를 수행
             거짓이면 실행되지 않고 넘어감
                
                if (조건식) {
                    참일시 실행;
                }
