CSS 블록과 스타일 시트
    HTML 요소에 스타일을 지정하는 항목을 말한다.
        CSS 블록은 선택자로 시작하며, 중괄호로 묶는다
    스타일 시트는 CSS블록들의 집합이다.
        스타일 시트는 <head> 요소 안에 스타일 요소 작성후 그 안에 시트 적용
    스타일 시트
        구조
            <head>
            <style>
            CSS 선택자 { 
                스타일 시트
            }
            </style>
            </head>

        <link>
            <link> 요소는 <head> 요소 안에 들어간다
            HTML 문서와 외부 리소스의 관계를 명시하는 HTML요소
            스타일 시트를 연결하기위해

            rel - HTML 문서와 연결할 외부 리소스의 관계를 지정
            href - 연결할 외부 리소스의 URL을 지정

    CSS 은 HTML이 해석된 화면에 출력하기 위한 (DOM) 문서객체모델을 생성해 출력하면 
    각 객체들을 화면에 출력전 CSS에 작성한 스타일을 적용시켜 출력한다

    결론: CSS는 HTML 문서에 있는 요소가 아닌 출력전 DOM 안에 각 객체에 적용된다



    CSS 선택자
       
        중괄호 앞에 적용된다
        CSS 블록에 지정한 스타일을 적용할 HTML 요소를 선택
        CSS 선택자는 스타일 시트 뿐만 아니라 JavaScript 또한 사용된다
           
            전체 선택자
                * 
                    DOM에 있는 모든 객체에 적용(PC에 부담을 많이 주기때문에 쓰지않는다)

            태그 선택자
                선택자 자리에 태그의 이름을 작성
                    작성한 태그에 해당하는 DOM 객체를 적용

            아이디 선택자
                불러올때 #로 시작 - id속성
                id 속성
                 id="" 으로 지정하며 단한개의 id만 존재한다

                불러올때 .으로 시작 - class속성
                    작성한 id, class인 속성을 가져옴
                class 속성
                 class=""로 지정하며 여러 HTML 요소에 같은 속성값을 지정
                 일정 요소<html>,<head>,<meta>등을 제외한 모든요소에 사용가능
                
                id, class 속성 은 객체마다 1개만 쓸수있다 2개를 사용할경우 뒤에 있는 속성이 사라짐
                
            클래스 선택자
                선택자.선택자 - 두 선택자로 지정된 값을 적용
                    두개의 선택자에 해당하는 DOM객체를 가져옴
                    교집합인 선택자를 가져옴

        상대 선택자
         두 선택자가 이어져있는 형태로 나옴
            후손 선택자
                A B
                    A 선택자로 선택된 요소의 후손 요소(뒤에 나오는 모든 요소)들중 B를 적용
            자식 선택자
                A > B 
                    A 선택자로 선택된 요소의 자식 요소(바로 다음 요소)들중 B를 적용
            인접 형제 선택자
                A + B
                    A 선택자로 찾은 요소의 바로 다음에 나오는 B를 선택
            일반 형제 선택자
                A ~ B
                    A 선택자로 찾은 요소의 다음 나오는 모든 형제요소들중 B를 선택
       
        속성 선택자
            [속성의 이름]
                속성의 이름이 지정된 HTMl 요소를 찾는다
            [속성의 이름=값]
                속성의 이름,값이 지정된 HTMl 요소를 찾는다
                보통 input 요소랑 같이 쓴다
                
        가상 클래스 선택자
             선택자에 추가하는 키워드로, 선택한 HTML 요소가 특정한 상태여야 선택된다
             가상 틀래스 선택자는 ":"으로 시작한다

                반응 선택자
                 사용자의 마우스 동작에 의해 생성되는 특정한 상태에 있는 요소를 선택
                    :active
                        마우스 버튼을 누르고 있는 동안 적용
                    :hover
                        마우스 버튼을 올려두는 동안 적용

                상태 선택자
                 입력 요소의 상태의 따라 요소를 선택
                    :checked
                        input 요소중 타입이 checkbox이거나 radio 인 요소들이 선택될때 적용
                    :enabled
                        활성화(선택이 가능하고 작성도 가능한 상태)된 상태인 요소들만 적용
                    :disabled
                        비활성화(선택도 못하고 작성도 못하는 상태)된 상태인 요소들만 적용
                    :focus
                        입력 요소들이 입력가능 상태인 요소들을 적용

                구조 선택자
                 형제관계에 있는 HTML 요소들중 특정 위치에있는 HTML 요소를 선택
                 주의할점 따지고 보면 2개의 태그를 붙여서 작성하는거기에 교집합에 해당하는 태그 요소를 선택해야한다
                    :first-child
                        형제 관계가 있는 요소들 중 첫번째에 있는 요소를 선택
                    :last-child
                        형제 관계가 있는 요소들 중 마지막에 있는 요소를 선택
                    :nth-child(수열)
                        형제 관계가 있는 요소들 중 앞에서 수열번째 있는 요소를 선택
                    :nth-last-child(수열)
                        형제 관계가 있는 요소들 중 뒤에서 수열번째 있는 요소를 선택

                            여기서 수열에는 식을 넣을수 있다
                            기본값은 n 이며 0부터 1씩 증가하는 의미다
                            즉 (n) = 1,2,3,4,... 이라는 뜻이고
                            (3n+1) = 1,4,7,10,13,... 이라는 뜻이다(3 * 0 + 1),(3 * 1 + 1)
                            그래서 수열을 작성할때 odd라고 작성하면 홀수 even이라고 작성하면 짝수를 의미한다     

                형태 구조 선택자
                 태그 형태를 구분해서 형제요소들 중에서 특정 위치에 있는 HTML 요소를 선택
                    :first-of-type   
                        앞에 붙인 태그 요소의 형제관계중 같은 태그를 가진 첫번째 요소 
                    :last-of-type    
                    :nth-of-type(수열)
                    :nth-last-of-type(수열)

                링크 선택자
                 href속성이 설정된 <a> 요소에 적용되는 선택자
                    :link
                        앵커 요소중 href 속성을 가지고있는 링크를 선택
                    :visited
                        앵터 요소중 herf 속성을 가지고있고 방문했었던 링크를 선택

                부정 선택자
                 주어진 선택자로 선택되지 않는 HTML요소들을 선택
                    :not(선택자)
                        괄호 안에있는 선택자를 제외한 나머지를 선택

                가상요소 선택자
                 실제 HTML요소가 아닌 특정 영역을 HTML 요소인것처럼 선택
                 이러한 영역을 가상요소 라고한다
                 가상요소 선택자들은 '::'을 붙인다
                    ::first-letter
                        HTML 요소의 내부 텍스트 중에 첫번째 문자를 선택
                    ::first-line
                        HTML 요소의 내부 텍스트 중에 첫번째 줄에 있는 텍스트를 선택

                전후 문자 선택자
                 지정한 요소의 첫번째 자식 요소 이전이나 마지막 요소 이후에
                 가상요소응 자식요소로 추가해준다
                    ::before
                        HTML 요소의 실제 내용 앞에 생성되는 자식요소를 생성
                    ::after
                        HTML 요소의 실제 내용 뒤에 생성되는 자식요소를 생성

                반응 문자 선택자
                 사용자가 문자와 반응해서 생기는 영역을 선택
                    ::selection
                        사용자가 드래그한 영역을 요소인것처럼 선택



    CSS 스타일 속성
     스타일 속성은 명시적으로 설정하지 않으면 해당 스타일 속성의 기본값이 적용된다
     각 스타일 속성은 개별적인 스타일을 지정한다
     콜론(';')으로 구별한다
     CSS 블록에는 스타일 속성이 여러 개 들어갈 수 있다
     각각의 스타일 마다 설정할수있는 정해진 값이 존재한다
        
        글자 단위(훨씬 많음)
            px  - 픽셀( 컴퓨터 그래픽에서 기본 값 )
            pt  - 포인트
            %   - 백분율 단위
            em  - 배수 단위( 현재 HTML 요소의 글자의 크기를 기준 )
            rem - 배수 단위( root요소 즉, <html> 요소의 글꼴 크기를 기준 )
            vh,vw - 브라우저 화면 높이나 너비의 백분율

        색상 단위(훨씬 많음)
            black  - 색상 이름
          #000000 - HEX 코드 단위(16진수)
            rgb - RGB 색상 단위(10진수)
            rgba - RGBA 색상 단위(RGB + 투명도)
            hsl - HSL 색상 단위(색상, 채도, 명도)
            hsla - HSLA 색상 단위(HSL + 투명도)



    CSS 스타일의 상속
     HTML 요소에 설정한 스타일 속성이 그 요소의 후손 요소들에게 영향을 미치는것
     font, color, text-aligh 등이 있다
     각 스타일은 반드시 정해져있는 값으로만 결정할수있다

        inherit 속성 값, initial 속성 값
         이 두 속성은 모든 스타일 속성에서 사용한다

            inherit - 해당 스타일 속성의 값을 부모 요소에게 상속받아 설정한다
            initial - 해당 스타일 속성의 기본 값으로 설정한다
    
    각 요소는 뷰 포트에 표시되며 크게 4 영역으로 볼수있다

    1. 외부 여백 margin
    2. 테두리 border
    3. 내부 여백 padding
    4. 콘텐트 영역 content box

    각 영역의 크기를 조정하기위한 속성과 기본 값
         
    콘텐트 영역

        width    - 콘텐트 영역의 너비를 설정   -  auto
        height   - 콘텐트 영역의 높이를 설정   -  auto
          Length - 크기 단위를 사용해 설정
          percentage - 컨테이닝 블록 너비 또는 높이의 백분율로 설정
         
    테두리, 외부, 내부 여백

        margin  - 외부 여백의 크기를 설정    -  0
        border  - 태두리의 크기를 설정       -  0
        padding - 내부 여백의 크기를 설정    -  0
         
         크기 지정 순서 위 오른쪽 아래 왼쪽 순이다
        ex)  margin: 10px 20px 30px 40px;
         마주보는 값이 같으면 생략 가능
        ex)  margin: 10px 30px 10px 30px;
        =    margin: 10px 30px;
         값이 3개일때는 무조껀 좌,우가 같을때만 3개로 생략가능(왼쪽이 생략)
        ex)  margin: 10px 20px 30px 20px;
        =    margin: 10px 20px 30px;
         물론 각각 따로 지정할수있다
         margin-(top, right, bottom, left)
        ex)  margin-top: 10px;

         Length - 크기 단위를 사용해 설정
         percentage - 컨테이닝 블록 너비의 백분율로 설정
    
    테두리, 배경을 표현하는 스타일 속성
        
        border-width 스타일 속성
         테두리의 두께를 지정
            medium - 중간두께의 테두리를 설정(기본 값, 3px)
            thick - 두꺼운 테두리를 설정
            thin - 얇은 케두리를 설정

        border-style 스타일 속성
         테두리의 외형을 지정
            none - 그리지 않음(기본 값)
            hidden - 테두리가 있지만 보이지 않음
            dotted - 점
            dashed - 점선선
            solid - 선
            double - 2줄
            groove, ridge, inset, outset 등등

        border-color 스타일 속성
         테두리의 색상을 지정

        border 스타일 속성
         border 스타일은 설정하는 속성은 여러 값이 들어갈 경우
         들어가는 순서는 상관없고 설정하지않을경우 생략가능하다
         물론 설정하지않을경우 기본값이 설정된다
        
        border-radius 스타일 속성
         HTML 요소의 모서리를 둥근 형태로 만들기 위한 원의 반지름을 지정
         (모서리가 둥근 사각형, 원을 만들 수 있다)(기본 값 - 0)
         지정하는 순서는 왼쪽 상단, 오른쪽 상단, 오른쪽 하단, 왼쪽 하단 순이다
         ex)
         border-radius: <1> <0> <10> <5>;
         물론 마주보는 값이 같으면 생략 가능하다
         border-radius: <1> <0> <1> <2>;
         border-radius: <1> <0> <2>;

        

    HTML 요소의 높이와 너비
    너비 - 너비는 그 요소의 조상 요소의 컨텐트 영역이 기본 영역이 된다
    높이 - 높이는 그 요소의 컨텐트 요소에 높이에 따라 결정된다
        전체 높이 = height + padding-top + padding-bottom + border-top + border-bottom
        전체 너비 = width + padding-right + padding-left + border-right + border-left

    box-sizing 스타일 속성
     width 속성과 height 속성이 어떤 영역의 크기를 설정할 것인지를 지정 

        content-box - width 속성과 height 속성이 콘텐트 영역의 크기를 지정(기본 값)
        border-box  - width 속성과 height 속성이 테두리를 포함한 영역의 크기를 지정

     ()안에 원하는 수식을 넣으면 해당 수식을 계산해서 표현한다
     

    배경(background)
        background-color - 배경색을 지정(테두리영역까지)
            기본값(transparent) - 투명한 색상을 지정
        background-image - 배경에 넣을 이미지를 지정(padding-box에 지정)
            기본값(none) - 배경 이미지를 지정하지 않음
            url() - 배경 이미지의 UPL 지정
             쉼표로 구분해서 여러개의 이미지를 지정할수있다
             먼저나온 이미지가 위로 올라온다(종이가 포개진다고 생각)
        background-repeat - 배경이미지가 배경보다 작을 경우, 이미지를 반복해서 지정
            기본값()
            no-repeat - 반복하지 않는다
            repeat-x - 오른쪽(x축)방향으로 반복해서 지정
            repeat-y - 아래쪽(y축)방향으로 반복해서 지정
        background-size - 배경 이미지의 크기를 설정
            기본값(auto) - 원본 이미지 크기로 지정
            Length - 크기 단위로 지정(너비 높이)
            precentage - HTML 요소의 크기의 백분율로 지정
            cover - 요소의 배경 영역을 가득 채우도록 지정(이미지가 화면에서 잘려도 1장으로 채운다)
            content - 요소의 배경 영역에 들어가도옥 지정(이미지가 작아도 화면안에 다 들어가도록 여러장을 사용)
        background-attachment - 배경이미지를 어떤 방식으로 브라우저 화면에 넣을 것인지 지정
            기본값(scroll) - 요소를 따라서 스크롤 되도록 지정
            fixed - 화면을 따라서 스크롤 되도록 지정(화면을 내려도 같은 위치에서 따라온다)
            local - 컨텐트를 따라서 스크롤 되도록 지정
        background-position - 배경 영역의 좌상단을 기준으로 배경 이미지의 위치를 지정
            기본값(0% 0%)
            pos - 크기 단위로 배경 이미지의 위치를 지정
            left-top - 좌상단에 위치를 지정
            left - 좌측중앙에 위치를 지정
            left-bottom - 좌하단에 위치를 지정
            right-top - 우상단에 위치를 고정
            right-bottom - 우하단에 위치를 고정
            center-center - 중앙에 위치를 고정
            ... 대충보면 견적 나오지?
        background-clip - 배경색을 넣을 영역을 지정
        background-origin - border 박스 안에서 어떤 여백, 영역을 기준으로 이미지를 배치할지 지정

        background 스타일 속성

        font-size 스타일 속성(부모 요소의 글자 크기)
         글자의 높이를 지정
            size - 크기 단위로 글자 크기 지정
            ...
            large - 크게 글자 크기 지정
            medium - 중간 정도의 글자 크기 지정
            smaller - 작은 글자 크기 지정 

        font-family 스타일 속성
         글꼴을 지정하는 스타일 속성
         글꼴의 이름으로 속성을 지정
         (여러단어로 이루어진 이름은 따옴표로 묶는다"")
         (다른 사용자도 그 글꼴이 있다고 장담할수 없으므로 여러 글꼴을 같이 지정)
         마지막에는 반드시 기본 글꼴를 지정(아래 3개가 기본 글꼴이다)
          serif - 명조체
          sans-serif - 고딕체
          monospace - 고정 폭 글꼴

        font-style 스타일 속성
         글자의 기울기를 지정
            normal - 보통 글자로 지정(기본값)
            italic - 기울어진 글자로 지정
            oblique - 완만하게 기울어진 글자로 지정(지원하지않는 글꼴이 많음)

        font-weight 스타일 속성
         글자의 두께를 지정
            normal - 보통 두께로 지정(기본값)
            bold - 굵은 글씨체로 지정
            bolder - 부모 요소보다 더 굵은 글씨체로 지정
            lighter - 부모 요소보다 얇은 글씨체로 지정

        font-variant 스타일 속성
         글자의 변형을 지정
            normal - 일반글자로 지정(기본값)
            
        line-height 스타일 속성
         행의 높이를 지정하는 스타일 속성 기본값은 HTML요소의 적용된 글꼴에 따라 적용
         텍스트는 행의 중간에 배치되기에 HTML 요소에서 내부 텍스트를 수직 정렬할 때 사용
         (글짜 위, 아래 여백과 글씨 높이을 지정한다고 보면 됨)
         (작성시 '{ line-height: 1.2; }' 라는건 현재 글자 크기의 1.2배 라는거다)
         그래서 1.5 = 1.5em = 150% 는 다 같은 말이다

          그래서 line-height같은 경우 size 뒤에 / 를 작성후 작성되야한다

         (-) 음수로도 표현하는데 그럴 경우 진행방향의 반대방향으로 지정된다
            normal - 보통 자간으로 지정(기본값)

            left - 왼쪽 정렬(기본값)

            relative - 문서의 일반적인 흐름에따라 배치한

         HTML요소의 크기나 위치는 컨테이닝 블록에 영향을 받으며 속성 값에 따라 컨테이닝 블록이 달라진다
            static, relative - 그 요소와 가장 가까운 블록 레벨 조상 요소의 콘텐트 영역
            absolute - 조상 요소중 가장 가까운 위치 지정 요소의 padding을 포함한 영역이거나

            fixed - 브라우저 화면(뷰 포트)

         부유하는 요소는 다른 부유하는 형형 제요소의 형제가 될수 없다

 그리고 배치되는 방향의 수직을 이루는 축을'cross axis'라 하며 위에서 아래로 진행
 'cross axis'가 시작하는 점이'cross start' 라 하고 끝나는 점을 'cross end'라 한다
 아이템 요소 의 진행방향과 평행하는 크기가 'main size', 수직되는 크기는'cross size'라 한다


    컨테이너 요소들에 적용하는 스타일 속성 

        display 스타일 속성

        flex 스타일 속성
         요소를 블록 또는 인라인 형식의 플렉스 컨테이너로 설정
        
        flex-direction 스타일 속성
         아이템들을 배치할 기본축의 방향을 설정
            row - 왼쪽부터 오른쪽으로 (기본값)
            column - 위에서 아래로 
            row, column-reverse - 반대 방향으로
       
        flex-wrap 스타일 속성
         모든 아이템 요소들을 한 행에 배치할 여유 공간이 없을때 중 바꿈을 랗 것 인지 설정
            nowrap - 공간이 없으면 최대한 너비를 줄임 (기본값)
            wrap - 자기 크기대로 설정 자리가 없으면 아래로 내려감
            wrap-reverse - 자기 크기대로 설정 자리가 없으면 위로 올라감

        flex-flow 스타일 속성
         위에 속성을 한번에 설정하는 속성

        justify-content 스타일 속성
         기본 축 상에서 아이템 요소들의 배치 기준점과 여백을 설정
            flex-start(normal) - 왼쪽부터 차례대로 지정 남는 공간이 있어도 채우지 않음(기본값)
            flex-end - 오른쪽으로 붙음
            center - 가운데로 모임
            space-between - 시작되는 요소는 main start쪽에 붙고 마지막요소는 main end 쪽에 딱 붙음 사이에 있는 요소들의 간격은 일정하게 설정
            space-around - 아이템 요소들의 갯수만큼 기본 축을 나누고 나눈 한 가운데 요소를 배치(여백, 위치가 동일)
            space-evenly - main start와 요소의 간격, 요소와 요소의 간격, 요소와 main end의 간격을 동일하게 설정

        align-items 스타일 속성
         교차 축 상에서 한 행으로 배치된 아이템 요소들의 정렬 방법을 설정
            flex-start - 위에 설명과 같지만 수평이 아닌 수직
            flex-end - 동일하다
            center - 동일하다
            stretch - 아이템 요소들의 cross size를 높이만큼 늘림(기본값)
            baseline - 모든 아이템 요소들의 중싱을 1개로 맞추어서 설정

        align-content 스타일 속성
         교차축상에서 여러 행으로 배치된 아이템요소들의 배치 기준점과 여백을 설정
            flex-start - cross line 과 관련됨 설명은 위와 비슷
            flex-end - cross line 과 관련됨 설명은 위와 비슷
            center - cross line 과 관련됨 설명은 위와 비슷
            stretch - 위에 설정과 비슷하지만 여러줄일때 사용(기본값)
            space-between - cross line 과 관련됨 설명은 위와 비슷
            space-around - cross line 과 관련됨 설명은 위와 비슷
            space-evenly - cross line 과 관련됨 설명은 위와 비슷

        아이템 요소들의 정렬
         justify 라고 나오면 기본축 설정
         align 이라고 나오면 교차축 설정

         -content 여러줄을 정렬축으로 배치
         -item 요소들이 정렬축으로 한불로 배치

         gap 스타일 속성
          아이템 요소들 사이의 간격을 설정(컨테이닝 블러 안 요소)
          아이템 요소들이 행과 열로 이루워져있다면 간격을 따로 설정가능
          row-gap, column-gap 으로 사용

    아이템 요소에 설정하는 스타일 속성
        (규칙적이지 않아서 선호하지 않음)

웹 페이지 레이아웃
 웹 페이지의 구성 요소들을 효과적으로 정리하고 배치하는 것
 대부분의 웹 페이지는 큰틀에서 보면 상당히 비슷한 레이아웃을 가진다
 헤더, 내비게이션, 사이드 바, 본문, 푸터 등으로 구분할수있다

    헤더(header)
    내비게이션(navigation)
    사이드 바(sidebar) 본문(content)
    푸터(footer)
