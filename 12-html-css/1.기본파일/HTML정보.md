HTML(Hyper-Text Markup Language)
    
    html 문서의 구조와 콘텐츠의 근간
    html 문서를 작성하기위한 지배적인 마크업 언어
    꺽은 괄호 사이에 입력 < 태그 > 으로 구성
    태그는 소분자로만 사용
    html 과 css 로 작성한 문서는 정적 문서이기에 수정이 불가하다
    html 문서는 <html>요소 로 시작해 <html>요소로 끝난다


HTML 요소
    
    HTML 요소의 역할
     
        HTML 문서의 정보 등을 표현
     
            <head> 요소에 '반드시' 들어가야하는 요소들
                <meta>, <title>
     
            <head> 요소의 포함되는 요소들
                <meta>, <title>, <link>, <style>, <script>
            
            <body> 요소에 들어가는 내용
                html 요소와 컨텐츠를 표현하는 요소

    
    HTML 태그의 종류

        문서의 정보를 표현 하는 요소들들
            <meta>, <title>,<link>,<style>,<script>

                <meta>
                    문서의 메타데이터 등을 표현
                    문서에서 사용하는 문자의 인코딩을 지정해줘야함
                    기본 

                <title>
                    문서의 제목을 표현(제목이 없는 문서는 있을수 없으니 반드시 지정해야함)
                    

        콘텐츠를 표현하는 요소들

            글자 태그

                제목(h4 까지는 사용되지만 5부터는 거의 사용되지 않는다)
                    <h1>,<h2>,<h3>,<h4>,<h5>,<h6>
                        제목(heading)을 표현
                본문
                    <p>,<br>,<hr>
                        <p>  - 단락(paragraph)을 표현
                        <br> - 텍스트의 강제 개행(break)를 표현
                        <hr> - 가로 구분선(horizontal break)을 표현
                앵커
                    <a>
                        문서와 문서의 연결을 표현(클릭시 이동하는 것, 클릭시 페이지가 넘어가는것 등등등)
                        <a href="이동할 페이지의 주소">지정할이름</a> - 입력해둔 페이지로 이동
                        <a href="#">지정할이름</a> - 문서의 최상단으로 이동
                        <h2 id="target">p1</h2>
                        ...
                        <a href="#target">누르면 이동</a> - id를 지정한곳으로 이동
                            id 속성 class속성
                                영문자 숫자 - _ 만 조합해서 작성
                                공백문자나 다른 문자를 포함할수없다
                                첫글자는 영문자로 시작
                                식별자는 대,소문자를 구분한다
                                id속성은 단 하나만 존재할 수 있다
                                    id 각각의 요소에 개별적인 식별자를 부여하기위한 속성
                                        단하나의 요소만 사용가능능
                                    class 여러 요소를 묶어 식별자를 부여하기위한 속성
                                        여러개의 속성을 빈칸으로 구분하여 나열한다다
                글자 형태
                    <b>,<i>,<strong>,<em>,<small>,<sub>,<sup>,등등
                        css를 사용하기에 요즘은 이런 태그들을 사용하지않는다

            문자 인코딩과 문자 집합

                문자 인코딩
                    사용자가 입력한 문자나 기호응 컴퓨터가 이해

                문자 집합

                ASCII(미국정보보안표분부호)
                    영문 알파벳과 숫자 부호만 표현할수있도록 설계된 컴퓨터 언어

                유니코드(UTF-8,UTF-16)
                    전세계 언어를 컴퓨터 언어로 표현할수있도록 해주는 컴퓨터 언어
                    완성현 한글 11,1172 자가 들어있다

                한글 문자 집합
                    EUC-KR
                    CP949
                    유니코드

        문서의 구조를 표현하는 태그 

            콘텐츠를 표현하는 태그

                목록
                    <ul>, <ol>, <li>, <dl>, <dt>, <dd>

                        <ul> - 순서가 없는 목록
                        <ol> - 순서가 있는 목록
                          <li> - <ul>, <ol> 안에만 들어가는 요소소 목록안에있는 각각의 항목
                        
                        <dl> - 정의목록(표제어)을 표현
                          <dt> - 표제어를 정의하는 용어를 표현 
                          <dd> - 용어의 설명을 표현

                표
                    <table>, <thead>, <tbody>, <tfoot>, <tr>, <td> 
                        <table> - 표를 표현 
                        ㄴ> 가장 복잡한 콘텐츠 작성, 화면에 출력도 어려움 반드시 필요할때만 사용
                        보통 아래 요소들은 하나씩 들어간다.

                          <thead> - 표의 제목 행들의 묶음(생략가능)
                          <tbody> - 표의 본문 행들의 묶음(생략불가)
                          <tfoot> - 표의 푸터 행들의 묶음(생략가능)

                            <tr> - 표의 행(row)을 표현
                              <th> - 표의 제목 셀을 표현(굵은 글씨체로 중앙정렬)
                              <td> - 표의 본문 셀을 표현
                              표의 병합은 2가지 row, col 이다 사용법은 해당 날짜

                이미지
                    <img>
                        브라우저 화면에 이미지를 표시
                        <img> 태그는 시작 태그만 있는 HTML 태그이다
                        <img src="" alt="" width="" heigth="">
                            사용하지 않는다면 적지않아도 되며 이미지는 원본비율을 맞추거나 너비, 높이 둘중 하나만 사용해야한다
                            하지만 src 를 사용한다면 alt 또한 사용해야한다

                            속성
                                src - 이 요소에서 표시할 이미지의 경로를 지정
                                ㄴ> 사용할 이미지가 폴더안에 있을경우 "폴더이름/이미지.jpg" 이런식으로 사용
                                
                                alt - 이미지가 나오지 않을때 대체 텍스트(상세히 적어야함)
                                
                                width - 이미지의 너비지정
                                ㄴ> width="숫자"
                                heigth - 이미지의 높이지정
                                ㄴ> heigth="숫자"

                입력양식
                    <form>
                    ㄴ> <form  action="어디로 보낼것인가" method="어떻게 전달받을건가">
                    </form>
                        속성
                            action - 입력 데이터를 전달달받아 처리할 웹 에플리케이션의 URL을 지정 
                            /기본값 - 현재 URL
                            method - 입력 데이터를 전달하는 방식을 지정 
                            /기본값 - get

                        전송방식
                            get
                             입력 데이터를 URL에 쿼리스트링 형태로 붙여서 전송
                             테스트나 디버깅이 쉽고 URL에 데이터가 보이므로 북마크가 가능하다
                             URL에 길이제한이 있고 보안이 취약해 대용량 데이터 전송에 불리하다
                            post
                             데이터를 HTTP 에 담아서 전송
                             데이터 길이제한이 없고 암호화가 가능해 상대적으로 안전
                             URL에 정보가 없어 북마크 불가 GET 보다 속도,캐싱 면에서 불리할 수 있다

                        입력 요소
                            <input>
                                사용자가 데이터를 입력할 수 있는 입력 영역을 지정
                                type 속성을 지정해 다양한 형태로 사용
                                (type, name 속성은 필수로 지정해야한다)
                            <select>
                                선택 입력 요소
                             <option>
                                 선택 입력 요소의 항목 
                             <optgroup>
                                 선택 입력 요소 항목의 그룹
                            <textarea>
                                여러 줄을 입력할 수 있는 글 상자
                            <button>
                                버튼 입력 요소
                            <label>
                                입력 요소의 캡션
                            <fieldset>
                                입력 요소의 그룹
