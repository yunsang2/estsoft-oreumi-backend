상대 선택자
         두 선택자가 이어져있는 형태로 나옴
            후손 선택자
                A B
                    A 선택자로 선택된 요소의 후손 요소(뒤에 나오는 모든 요소)들중 B를 적용
            자식 선택자
                A > B 
                    A 선택자로 선택된 요소의 자식 요소(바로 다음 요소)들중 B를 적용
            인접 형제 선택자
                A + B
                    A 선택자로 찾은 요소의 바로 다음에 나오는(진행방향) B 요소를 선택
            일반 형제 선택자
                A ~ B
                    A 선택자로 찾은 요소의 다음 나오는 모든 형제요소들중 B 요소를 선택
       
        속성 선택자
            [속성의 이름]
                속성의 이름이 지정된 HTMl 요소를 찾는다
            [속성의 이름=값]
                속성의 이름에 값이 지정된 HTMl 요소를 찾는다
                보통 input 요소랑 같이 쓴다
                
        가상 클래스 선택자
             선택자에 추가하는 키워드로, 선택한 HTML 요소가 특정한 상태여야 선택된다
             가상 틀래스 선택자는 ":"으로 시작한다
             ex) li:active {...}

                반응 선택자
                 사용자의 마우스 동작에 의해 생성되는 특정한 상태에 있는 요소를 선택
                    :active
                        마우스 버튼을 누르고 있는 동안 적용
                    :hover
                        마우스 버튼을 올려두는 동안 적용

                상태 선택자
                 입력 요소의 상태의 따라 요소를 선택
                    :checked
                        input 요소중 타입이 checkbox이거나 radio 인 요소들이 선택될때 적용
                    :enabled
                        활성화(선택이 가능하고 작성도 가능한 상태)된 상태인 요소만 적용
                    :disabled
                        비활성화(선택도 못하고 작성도 못하는 상태)된 상태인 요소만 적용
                    :focus
                        입력 요소들이 입력가능 상태인 요소를 적용

                구조 선택자
                 형제관계에 있는 HTML 요소들중 특정 위치에있는 HTML 요소를 선택
                 %주의할점% 따지고 보면 2개의 태그를 붙여서 작성하는거기에 교집합에 해당하는 태그 요소를 선택해야한다
                    :first-child
                        형제 관계가 있는 요소들 중 첫번째에 있는 요소를 선택
                    :last-child
                        형제 관계가 있는 요소들 중 마지막에 있는 요소를 선택
                    :nth-child(수열)
                        형제 관계가 있는 요소들 중 앞에서 (수열)번째 있는 요소를 선택
                    :nth-last-child(수열)
                        형제 관계가 있는 요소들 중 뒤에서 (수열)번째 있는 요소를 선택

                            여기서 수열에는 식을 넣을수 있다
                            기본값은 n 이며 0부터 1씩 증가하는 정수라는 의미다
                            즉 (n) = 0,1,2,3,4,... 이라는 뜻이고
                            (3n+1) = 1,4,7,10,13,... 이라는 뜻이다(3 * 0 + 1),(3 * 1 + 1)..
                            그래서 수열을 작성할때 odd라고 작성하면 홀수 even이라고 작성하면 짝수를 의미한다     

                형태 구조 선택자
                 태그 형태를 구분해서 형제요소들 중에서 특정 위치에 있는 HTML 요소를 선택
                    :first-of-type   
                        앞에 붙인 태그 요소의 형제관계중 같은 태그를 가진 첫번째 요소 
                    :last-of-type    
                    :nth-of-type(수열)
                    :nth-last-of-type(수열)

                링크 선택자
                 href속성이 설정된 <a> 요소에 적용되는 선택자
                    :link
                        앵커 요소중 href 속성을 가지고있는 링크를 선택
                    :visited
                        앵터 요소중 herf 속성을 가지고있고 방문했었던 링크를 선택

                부정 선택자
                 주어진 선택자로 선택되지 않는 HTML요소들을 선택
                    :not(선택자)
                        괄호 안에있는 선택자를 제외한 나머지를 선택

                가상요소 선택자
                 실제 HTML요소가 아닌 특정 영역을 HTML 요소인것처럼 선택
                 이러한 영역을 가상요소 라고한다
                 가상요소 선택자들은 '::'을 붙인다
                
                문자 선택자
                 HTML 요소의 내부 택스트 중에서 지정한 조건에 해당하는 글자를 가상 요소로 지정해서 선택
                    ::first-letter
                        HTML 요소의 내부 텍스트 중에 첫번째 문자를 선택
                    ::first-line
                        HTML 요소의 내부 텍스트 중에 첫번째 줄에 있는 텍스트를 선택

                전후 문자 선택자
                 지정한 요소의 첫번째 자식 요소 이전이나 마지막 요소 이후에
                 가상요소응 자식요소로 추가해준다
                 content 속성으로 가상요소를 추가해야 해당 선택자가 적용된다
                    ::before
                        HTML 요소의 실제 내용 앞에 생성되는 자식요소를 생성
                    ::after
                        HTML 요소의 실제 내용 뒤에 생성되는 자식요소를 생성

                반응 문자 선택자
                 사용자가 문자와 반응해서 생기는 영역을 선택
                    ::selection
                        사용자가 드래그한 영역을 요소인것처럼 선택



CSS 스타일 속성
     스타일 속성은 명시적으로 설정하지 않으면 해당 스타일 속성의 기본값이 적용된다
     각 스타일 속성은 개별적인 스타일을 지정한다
     콜론(';')으로 구별한다
     CSS 블록에는 스타일 속성이 여러 개 들어갈 수 있다
     각각의 스타일 마다 설정할수있는 정해진 값이 존재한다
        
        글자 단위(훨씬 많음)
            px  - 픽셀( 컴퓨터 그래픽에서 기본 값 )
            pt  - 포인트
            %   - 백분율 단위
            em  - 배수 단위( 현재 HTML 요소의 글자의 크기를 기준 )
            rem - 배수 단위( root요소 즉, <html> 요소의 글꼴 크기를 기준 )
            vh,vw - 브라우저 화면 높이나 너비의 백분율

        색상 단위(훨씬 많음)
            black  - 색상 이름
          #000000 - HEX 코드 단위(16진수)
            rgb - RGB 색상 단위(10진수)
            rgba - RGBA 색상 단위(RGB + 투명도)
            hsl - HSL 색상 단위(색상, 채도, 명도)
            hsla - HSLA 색상 단위(HSL + 투명도)



    CSS 스타일의 상속
     HTML 요소에 설정한 스타일 속성이 그 요소의 후손 요소들에게 영향을 미치는것
     font, color, text-aligh 등이 있다
     각 스타일은 반드시 정해져있는 값으로만 결정할수있다

        inherit 속성 값, initial 속성 값
         이 두 속성은 모든 스타일 속성에서 사용한다

            inherit - 해당 스타일 속성의 값을 부모 요소에게 상속받아 설정한다
            initial - 해당 스타일 속성의 기본 값으로 설정한다
    
    각 요소는 뷰 포트에 표시되며 크게 4 영역으로 볼수있다

    1. 외부 여백 margin
    2. 테두리 border
    3. 내부 여백 padding
    4. 콘텐트 영역 content box

    각 영역의 크기를 조정하기위한 속성과 기본 값
         
    콘텐트 영역

        width    - 콘텐트 영역의 너비를 설정   -  auto
        height   - 콘텐트 영역의 높이를 설정   -  auto
          Length - 크기 단위를 사용해 설정
          percentage - 컨테이닝 블록 너비 또는 높이의 백분율로 설정
         


    테두리, 외부, 내부 여백

        margin  - 외부 여백의 크기를 설정    -  0
        border  - 태두리의 크기를 설정       -  0
        padding - 내부 여백의 크기를 설정    -  0
         
         크기 지정 순서 위 오른쪽 아래 왼쪽 순이다
        ex)  margin: 10px 20px 30px 40px;
         마주보는 값이 같으면 생략 가능
        ex)  margin: 10px 30px 10px 30px;
        =    margin: 10px 30px;
         값이 3개일때는 무조껀 좌,우가 같을때만 3개로 생략가능(왼쪽이 생략)
        ex)  margin: 10px 20px 30px 20px;
        =    margin: 10px 20px 30px;
         물론 각각 따로 지정할수있다
         margin-(top, right, bottom, left)
        ex)  margin-top: 10px;

         Length - 크기 단위를 사용해 설정
         percentage - 컨테이닝 블록 너비의 백분율로 설정
    


    테두리, 배경을 표현하는 스타일 속성
        
        border-width 스타일 속성
         테두리의 두께를 지정
            medium - 중간두께의 테두리를 설정(기본 값, 3px)
            thick - 두꺼운 테두리를 설정(5px)
            thin - 얇은 테두리를 설정(1px)

        border-style 스타일 속성
         테두리의 외형을 지정
            none - 그리지 않음(기본 값)
            hidden - 테두리가 있지만 보이지 않음
            dotted - 점선
            dashed - 긴 점선
            solid - 실선
            double - 2중 실선
            groove, ridge, inset, outset 등등

        border-color 스타일 속성
         테두리의 색상을 지정

        border 스타일 속성
         border 스타일은 설정하는 속성은 여러 값이 들어갈 경우
         들어가는 순서는 상관없고 설정하지않을경우 생략가능하다
         물론 설정하지않을경우 기본값이 설정된다
         ex) border: [ <border-width> || <border-style> || <border-color> ] | inherit;
        
        border-radius 스타일 속성
         HTML 요소의 모서리를 둥근 형태로 만들기 위한 원의 반지름을 지정
         (모서리가 둥근 사각형, 원을 만들 수 있다)(기본 값 - 0)
         지정하는 순서는 왼쪽 상단, 오른쪽 상단, 오른쪽 하단, 왼쪽 하단 순이다
         ex)
         border-radius: <1> <0> <10> <5>;
         물론 마주보는 값이 같으면 생략 가능하다
         border-radius: <1> <0> <1> <2>;
         border-radius: <1> <0> <2>;

        

    HTML 요소의 높이와 너비
    너비 - 너비는 그 요소의 조상 요소의 컨텐트 영역이 기본 영역이 된다
    높이 - 높이는 그 요소의 컨텐트 요소에 높이에 따라 결정된다
        전체 높이 = height + padding-top + padding-bottom + border-top + border-bottom
        전체 너비 = width + padding-right + padding-left + border-right + border-left

    box-sizing 스타일 속성
     width 속성과 height 속성이 어떤 영역의 크기를 설정할 것인지를 지정 

        content-box - width 속성과 height 속성이 콘텐트 영역의 크기를 지정(기본 값)
        border-box  - width 속성과 height 속성이 테두리를 포함한 영역의 크기를 지정

    배경(background)
        background-color - 배경색을 지정(테두리영역까지)
            기본값(transparent) - 투명한 색상을 지정

        background-image - 배경에 넣을 이미지를 지정(padding-box에 지정)
            기본값(none) - 배경 이미지를 지정하지 않음
            url() - 배경 이미지의 UPL 지정
             쉼표로 구분해서 여러개의 이미지를 지정할수있다
             먼저나온 이미지가 위로 올라온다(종이가 포개진다고 생각)

        background-repeat - 배경이미지가 배경보다 작을 경우, 이미지를 반복해서 지정
            기본값()
            no-repeat - 반복하지 않는다
            repeat-x - 오른쪽(x축)방향으로 반복해서 지정
            repeat-y - 아래쪽(y축)방향으로 반복해서 지정

        background-size - 배경 이미지의 크기를 설정
            기본값(auto) - 원본 이미지 크기로 지정
            Length - 크기 단위로 지정(너비 높이)
            precentage - HTML 요소의 크기의 백분율로 지정
            cover - 요소의 배경 영역을 가득 채우도록 지정(이미지가 화면에서 잘려도 1장으로 채운다)
            content - 요소의 배경 영역에 들어가도록 지정(이미지가 작아도 화면안에 다 들어가도록 여러장을 사용)

        background-attachment - 배경이미지를 어떤 방식으로 브라우저 화면에 넣을 것인지 지정
            기본값(scroll) - 요소를 따라서 스크롤 되도록 지정
            fixed - 화면을 따라서 스크롤 되도록 지정(화면을 내려도 같은 위치에서 따라온다)
            local - 컨텐트를 따라서 스크롤 되도록 지정

        background-position - 배경 영역의 좌상단을 기준으로 배경 이미지의 위치를 지정
            기본값(0% 0%) (x축 y축)
            pos - 크기 단위로 배경 이미지의 위치를 지정
            left-top - 좌상단에 위치를 지정
            left - 좌측중앙에 위치를 지정
            left-bottom - 좌하단에 위치를 지정
            right-top - 우상단에 위치를 고정
            right-bottom - 우하단에 위치를 고정
            center-center - 중앙에 위치를 고정
            ... 대충보면 견적 나오지?

        background-clip - 배경색을 넣을 영역을 지정

        background-origin - border 박스 안에서 어떤 여백, 영역을 기준으로 이미지를 배치할지 지정 padding-box 가 기본값
         ex) background-origin: border-box;

        background 스타일 속성
         background 여러값을 사용해서 한번에 지정이 가능하지만
         size와 position의 값은 position 값 뒤에 /를 사용해서 
         설정해야한다 물론 둘중 하나라도 사용할경우 둘다 작성해야한다



    HTML 요소의 공간 차지 형식(더 많음)
     HTML 요소가 브라우저 화면에 표시될 때 공간을 차지하는 방식

        종류
            블록 형식
                HTML 요소의 컨테이닝 블록 너비 전체를 상자형태로 차지
                안쪽 여백,바깥쪽 여백을 가진다
                스타일 속성으로 위치와 여백, 너비, 높이를 설정할수 있다
                <div>, <h>, <p>, 목록 요소, <form> 등등
            인라인 형식
                HTML 요소가 문장 내에서 텍스트처럼 영역을 차지
                줄을 바꾸지 않고 
                <a>, <span>, 글자 형식 요소 등등
            인라인 블록 형식
                인라인 형식의 특징을 기본으로 하며, 블록 형식의 특징을 같이 가진다
                문장 내에 텍스트처럼 영역을 차지하며, 위치를 설정할수없다

    CSS 가시 스타일 속성
        display
        HTML 요소의 표시형식을 설정
        기본값은 인라인 형식이다
            black - 크기를 지정할수있고 컨테이닝 블럭에 수평방향만큼의 공간을 차지한다
            inline - 크기를 지정할수없으며 컨테이닝 좌측 상단부터 나열된다
            inline-black - 인라인 형식을 가지지만 크기를 지정할수있는 블럭의 특징도 가지고있다
             인라인 요소에서는 위,아래 margin을 줄수 없고 크기또한 지정할 수 없다
             그래서 위 아래에 margin을 지정하려면 inline-black으로 표기하면 해결된다
            none - DOM 영역에 표기는 되지만 뷰포트 에는 사라지기때문에 다른 컨텐트들이 그자리를 차지한다 

        visibility
        HTML 요소의 표시 여부를 설정
            visible - 기본값(보인다)
            hidden - 숨김(보이지 않는다) 그 공간이 사라지는건 아니기에 다른 컨텐트들이 그 공간을 침범할수 없다

        opacity
        HTML 요소의 투명도를 설정
            1 - 기본값(최대 값)
            0 - 투명(투명 상태)

글자 스타일 속성
    CSS에서 글자 스타일 속성은 상속이 가능하다

        font-size 스타일 속성(부모 요소의 글자 크기)
         글자의 높이를 지정
            size - 크기 단위로 글자 크기 지정
            ...
            large - 크게 글자 크기 지정 (부모 요소의 글자 크기)
            medium - 중간 정도의 글자 크기 지정 (부모 요소의 글자 크기)
            smaller - 작은 글자 크기 지정 (부모 요소의 글자 크기)

        font-family 스타일 속성
         내부 텍스트의 글꼴을 지정하는 스타일 속성
         글꼴의 이름으로 속성을 지정
         (여러단어로 이루어진 이름은 따옴표로 묶는다"")
         (다른 사용자도 그 글꼴이 있다고 장담할수 없으므로 여러 글꼴을 같이 지정)
         마지막에는 반드시 기본 글꼴를 지정(아래 3개가 기본 글꼴이다)
          serif - 명조체
          sans-serif - 고딕체
          monospace - 고정 폭 글꼴

        font-style 스타일 속성
         글자의 기울기를 지정
            normal - 보통 글자로 지정(기본값)
            italic - 기울어진 글자로 지정
            oblique - 완만하게 기울어진 글자로 지정(지원하지않는 글꼴이 많음)

        font-weight 스타일 속성
         글자의 두께를 지정
            normal - 보통 두께로 지정(기본값)
            bold - 굵은 글씨체로 지정
            bolder - 부모 요소보다 더 굵은 글씨체로 지정
            lighter - 부모 요소보다 얇은 글씨체로 지정
