    viewport 메타 요소

        <meta> 요소
         HTML 문서의 메타데이터를 제공하기 위한 HTML 요소
         <head> 요소에 들어간다
        
        viewport 메타 요소
         브라우져 화면 설정과 관련된 정보를 제공한다
         모바일 웹 페이지에는 브라우저 화면의 너비를 기기의 화면 너비로 설정하는 viewport 메타 요소를 반드시 추가해야함

        width - 브라우저 화면의 너비("width=device-width")
        height - 브라우저 화면의 높이
        initial-scale - 초기 확대 비율
        user-scalable - 확대 및 축소 가능 여부
        등등

transition, animation 스타일 속성
 transition 스타일 속성은 단순한 움직임을 animation 스타일 속성은 복잡한 움직임을 만든다

transition, animation 영향을 받는 속성

    위치 속성: top, right, bottom, left
    크기 속성: width, height
    박스.테두리 설정: margin, padding, border-width, border-radius
    색상.투명도 설정: color, background-color, border-color, opacity
    변환 속성: transform



    transition 스타일 속성

        -duration 스타일 속성
         변형을 얼마에 걸쳐 진행할 것인지 설정
         transition 속성에 따라 변화가 생기면 해당 속성의 시간만큼 걸쳐서 변화함
         단위는 초(s) 또는 밀리 초(ms)
         기본값은 0

        -property 스타일 속성
          변형하고자 하는 스타일 속성을 따로 지정가능
          기본값은 all
          여서 스타일을 지정하고싶다면 , 로 구분하여 작성

        -delay 스타일 속성
         변형을 시작하기 전에 얼마나 지연할지 설정
         단위는 초(s) 또는 밀리 초(ms)
         기본값은 0

        -timing-function
         시간에 대해 변형이 진행되는 속도의 함수를 설정
         속성 값으로 함수 이름이나 cubic-bezier 함수를 지정한다
         기본값은 ease
            linear - 속도유지
            ease - 느리다가 빨라졌다가 느려짐
            ease-in - 초반에 느림
            ease-out - 후반에 느림
            ease-in-out - 빨랐다가 느렸다가 빨라짐

        

    animation 스타일 속성

        -duration
        -property
        -delay
        위에 속성은 transition 과 같이 동작한다

        -name 스타일 속성
         키 프레임으로 작성한 규칙을 불러올때 사용

        -iteration-count 속성
         애니메이션의 반복 횟수를 정수로 지정한다
         기본값은 1
         무한도 지정할 수 있다

        -direction 속성
         애니메이션이 반복될때 애니메이션의 진행 방향을 설정
         기본값은 nomal
         진행한 방향과 반대 방향은 alternate

        -play-state
         애니메이션의 재생 상태, 즉 일시 정지하거나 재생하도록 지정
         기본값은 runnig
         일시정지는 paused

        키 프레임 규칙
         animation 스타일 속성이 지정된 요소에 적용할 스타일 속성을 설정
         키 프레임 안에는 퍼센트 단위로 시간이 지났을때 적용할 CSS 블록을 작성한다
         작성시에는 같은 요소로만 작성해야한다 top,left만 사용하면 2개만 사용해야하고 중간에 bottom이 들어온다면 작동하지 않는다
         예외적으론 0% 와 100% 는 from, to 키워드를 사용함
         ex) @keyframes name {
                from{}
                25% {}
                50% {}
                75% {}
                to  {}
            }



    transform 스타일 속성과 2차원 변환
     HTML 요소를 변환하는 스타일 속성

        rotate()
         지정한 각도만큼 회전
         기본값은 0
         단위 deg

        scale()(X축과 Y축에 별도 함수 제공)
         지정한 배수만큼 확대 혹은 축소
         기본값은 1
         단위 scale

        translate()(X축과 Y축에 별도 함수 제공)
         지정한 크기만큼 이동
         단위 Length, percent(요소크기)

        skew()(X축과 Y축에 별도 함수 제공)
         지정한 각도만큼 기울임
         단위 deg

        -origin 스타일 속성
         해당 요소의 변환 중심을 지정
         중심을 지정하면 해당 중심을 기준으로 애니메이션이 진행


    
    transform 스타일 속성과 3차원 변환

        rotate3d()(X축과 Y축, Z축에 별도 함수 제공)
         지정한 각도만큼 회전

        scale3d()(X축과 Y축, Z축에 별도 함수 제공)
         지정한 배수만큼 확대 혹은 축소

        translate3d()(X축과 Y축, Z축에 별도 함수 제공)
         지정한 크기만큼 이동



 과 반응형 웹

    CSS 추가 규칙
     스타일 시트에서 앞에 @ 기호를 붙여서 만든 사용자 지정 규칙

        @keyframes 규칙
         위치 146

        @import 규칙
         스타일시트에서 다른 스타일 시트를 불러오는 추가 규칙
         (성능이 좋지않아 link 요소로 대체하는것이 좋다)

        @font-face 규칙
         사용자가 컴퓨터에 설치되지 않은 글꼴을 다운로드해서 사용하도록 하는 추가 규칙
         폰트 파일 용량 등의 문제로 인해 많이 사용하지 않는다
         (반드시 써야한다면 웹페이지에서 사용하도록 최적화된 웹 폰트를 사용하도록 한다)

         @media 규칙
          다양한 기기에서 HTML 문서가 적절한 형태로 출력되도록 하는 추가 규칙
          최근에는 HTML 문서 하나를 다양한 기기에서 사용하기 위해 부각되고있다
          미디어 퀴리로 기기를 구분하거나 기기의 화면 해상도와 화면 비율 등을 구분할 수 있다
          해당 규칙은 미디어 타입을 정해줘야한다
          ex)@media 미디어타입 {CSS}

            미디어 타입
                all - 모든 미디어 타입
                screen - 화면에 출력
                peint - 프린터로 출력
                등등

        
    
    반응형 웹
     가변 그리드, 가변 이미지, 미디어 쿼리를 이용해, 하나의 HTML 문서가 기기의 화면 너비에 따라 콘텐츠 구성이 적절하게 맞춰지도록 하는 기술
     이를 통헤, 사용자는 기기의 화면 크기에 상관 없이 서비스를 원할하게 사용할 수 있다
     많은 콘텐츠의 표현엔 적합하지 않다
     블로그, 회사 소개 페이지, 커뮤니티 등에 사용된다
     

        미디어 쿼리
         반응형 웹페이지를 만들기 위해 @media 추가 규칙으로 작성한 표현식
         미디어 쿼리를 사용하면, 콘텐츠를 변경하지 않고도 출력기기에 맞춰 출력할수 있다

        미디어 속성
         미디어 쿼이레서는 미디어 속성으로 화면 크기, 화면 비율 등을 구분한다
            width, height - 화면의 너비와 높이 (min, max)
            device-width, device-height - 기기의 너비와 높이 (min, max)
            등등
